# OU B.Tech Supply Exam Helper 🎓

A web app to help B.Tech engineering students pass their supplementary exams - focused on **Osmania University affiliated colleges**.

## 🌐 Live Demo
[Open App](https://karthikeyapodicheti.github.io/suppapp/)

## ✨ Features
- Semester-wise subject selection (1-8 semesters)
- Branch-specific subjects: **CSE, ECE, EEE, MECH, CIVIL, IT, AI&DS**
- For each subject:
  - 🎯 How to pass tips
  - 🔥 Most important topics
  - 📚 Unit-wise breakdown
  - 💡 Quick study tips
  - 📖 Recommended resources (textbooks + YouTube)

## 📚 Subjects Covered

| Semester | Subjects |
|----------|----------|
| 1-2 | M1, M2, M3, Physics, Chemistry, English, C Programming, Engineering Drawing, BEEE, Data Structures |
| 3-6 | Branch-specific core subjects |

### Branches
- **CSE**: DSA, DBMS, OS, COA, DAA, CN, SE, Compiler Design, ML, CNS
- **ECE**: Signals & Systems, ADE, EMTL, Digital Comm, MPMC, VLSI, DSP
- **EEE**: Electrical Machines, Electronic Circuit Analysis
- **MECH**: Theory of Machines, Strength of Materials, Thermodynamics
- **CIVIL**: SOM, Surveying, Fluid Mechanics
- **IT**: Java OOP, Web Technologies
- **AI&DS**: Python, ML, AI, Deep Learning, NLP, Big Data, Computer Vision

## 🚀 Usage
1. Open `index.html` in any browser
2. Select your semester
3. Select your branch
4. Click on a subject to view study guide

## 🛠️ Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
